# 1. if
```py
if 조건:
  코드
```
이런식으로 씀.
저 조건에 '10 > 5'와 '10 < 5'를 각각 넣어보자.

'10 > 5'는 참
'10 < 5'는 거짓

이기 때문에, 전자는 작동을 하고, 후자는 작동하지 않는다.

# 2. elif & else
if 조건문이 참일경우도 있고, 아닐경우도 있을것이다.
그리고 아닐 경우에만 어떤 코드를 넣고 싶을떄도 있을것이다.

이 문제는 else와 elif로 해결할 수 있다.

## 2.1. else
```py
if 조건:
  코드
else:
  코드
```
만약 조건이 거짓이라면, else에 있는 코드를 실행하게 된다.

```py
pw_correct = True

if pw_correct:
  print("출금")
else:
 print("비밀번호 틀림")
```
## 2.2. elif
그런데 조건을 더 추가 하고 싶을떄도 있을것이다.
그럴땐 elif를 쓴다.
```py
winner = 8

if winner > 10:
    print("Winner is greater than 10")
elif winner < 10:
    print("Winner is less than 10")
else:
    print("Winner is 10")
```

# 3. And & Or
## 3.1. input 받는법
```py
변수 = input('text)
```
기본적으로 하나의 변수만 입력 받고, string형으로 입력받음.

int형을 입력받고 싶으면
```py
변수 = int(input('text'))
```

## 3.2. and & or
* and: 두 조건이 참인 경우에 참을 반환.
* or: 두 조건중 1개 이상이 참인 경우 참을 반환.

False and False == False
False and True = False
True and False = False
True and True = True

False or False == False
False or True = True
True or False = True
True or True = True

# 5. while
while은 반복문임.
```py
while 조건문:
  코드
```
조건문이 참이면 while은 동작함.

예를 들어
```py
while True:
  print("hi")
```
이렇게 쓰면 True는 항상 참이기 때문에, 무한반복함.

## 5.1. 활용
내가 원하는 횟수만큼 하는 방법이 있다.
변수를 활용하면 된다. 변수를 하나 만들어서 반복문을 실행할때마다 값을 수정시켜서 반복문이 거짓이 되게 하면 된다.

예를 들면
```py
distance = 0

while distance < 20:
  print("거리 ->", distance)
  distance = distance + 1
```
이 코드를 보면.
distance는 0에서 시작하여 +1씩 증가하면서 19까지 값이 잘 출력된다.<br>
그러나 20이 되면 조건문이 거짓이 되어 while 루프를 빠져나오게 된다.
